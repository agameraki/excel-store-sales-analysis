# Store Sales Data Analysis (Excel)

## Project Overview
This project presents an end-to-end analysis of store sales data using Microsoft Excel. The objective of the analysis is to understand customer behavior, sales performance, and operational efficiency by transforming raw transactional data into actionable business insights.

The project demonstrates a structured data analysis workflow, including data cleaning, feature creation, and exploratory analysis using pivot tables and pivot charts. It is designed to showcase foundational data analysis skills suitable for entry-level data analyst or business analyst roles.

---

## Dataset Description
The dataset consists of order-level records with the following attributes:
- Customer demographics (gender, age, age group)
- Order date and derived order month
- Product category and size
- Sales channel
- Order status (delivered, cancelled, returned)
- Shipping state

This data enables multi-dimensional analysis across customer, product, regional, and channel-based perspectives.

---

## Key Business Questions Answered
This analysis aims to answer the following business-relevant questions:

- Which gender contributes more to overall sales?
- Which age groups show the highest purchasing activity?
- What is the distribution of order status, and how efficient is order fulfillment?
- Which states contribute the most to total orders?
- Which sales channels perform best in terms of order volume?
- Which product categories and sizes are most frequently purchased?

Answering these questions helps identify key customer segments, high-performing regions, and demand patterns.

---

## Data Cleaning & Preparation
Before conducting the analysis, the dataset was cleaned and prepared to ensure accuracy and usability. Key preparation steps included:
- Applying filters to review and clean the dataset efficiently
- Adjusting column widths and aligning data to improve readability
- Creating derived columns such as age groups and order months to support trend analysis
- Structuring the data to enable effective pivot table analysis

These steps ensured the data was analysis-ready and logically organized.

---

## Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Filtering and Formatting
- Exploratory Data Analysis (EDA)

---

## Analysis & Key Insights
The analysis revealed several meaningful patterns:
- Certain customer demographics contribute disproportionately to total sales
- A limited number of states act as primary revenue-driving regions
- Some sales channels significantly outperform others in customer engagement
- Specific product categories and sizes consistently show higher demand
- The majority of orders are successfully delivered, indicating strong operational performance

These insights can help businesses optimize marketing strategies, inventory planning, and channel partnerships.

---

## Conclusion
This project highlights how Excel can be effectively used for data analysis through structured preparation, pivot tables, and visual exploration. By answering key business questions and identifying actionable insights, the analysis demonstrates practical skills relevant to real-world data-driven decision-making.
